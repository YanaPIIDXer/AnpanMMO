# マスタの定義
マスタはエクセル（xlsx）形式で記述します。  
MasterDataディレクトリに保存してください。  
記述方法については他のマスタを参考にしてください。(タグについては後述します。)  

# マスタで使用可能するタグ
タグはＡ列に記述します。  
**$ITEM**  
必須。これより下に記述された項目がマスタデータとして出力されます。  
$ITEMタグの記述された二つ右上が変数名、一つ右上が変数の型になります。  
**$SERVER_ONLY**  
サーバにのみ出力したい場合に記述してください。  
**CLIENT_ONLY**  
クライアントにのみ出力したい場合に記述してください。  

# マスタ転送に関する注意点
転送先にMySQLがインストールされている必要があります。  
インストールしていない場合はインストールしていただくようお願いします。  
（と言うか、MySQLがないとそもそもサーバが起動できませんが・・・）  
また、Linuxに転送する場合、MySQLに接続するユーザ名・パスワードと同一のユーザを作成しておく必要がありますので注意してください。

# 転送対象の追加
マスタの転送先を転送するには、「転送対象の追加」ボタンを押下し、出てきたダイアログに以下の必要な情報を記述します。  
名称：転送先の名称。任意の名前を入力してください。  
ホスト：転送先のホスト。  
ユーザ名：SSH接続、及びMySQL接続に使用するユーザ名。  
パスワード：SSH接続、及びMySQL接続に使用するパスワード。  
バイナリ転送先ホスト：クライアント向けに出力するバイナリデータを転送するホスト名。（ここでも上記のユーザ名とパスワードを使用します。  ）
バイナリ転送先パス：クライアント向けに出力するバイナリデータを転送するパス。（大抵public_html以下にディレクトリを作成してそこに転送する事になると思います。）  

# マスタの転送
リストから転送先を選択し、「出力」ボタンを押下してください。  
マスタの転送が始まります。  

# マスタのデータベース名
マスタのデータベース名は**AnpanMMOMaster**です。  
あらかじめデータベースを作成の上、接続アカウントに権限を付与しておいてください。
