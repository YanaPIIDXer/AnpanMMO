# スクリプトモジュールのビルド＆デプロイ
プログラムで自動化しています。  
ScriptModuleディレクトリ内のSourceDeploy.exeを叩く事でソースコードがサーバ・クライアントにデプロイされます。  
※デプロイだけならバッチファイルだけで事足りるのですが、  
クライアント：先頭で対応するヘッダをincludeしなければならない。  
サーバ：先頭でstdafx.hをincludeしなければならない。  
と言う問題があったため、プログラムで対応しています。  

# スクリプトについて
実際にゲーム（或いはサーバ）上で実行されるスクリプトはGitでは無くSVNで管理します。（GitHubでオープンにはしません。）
