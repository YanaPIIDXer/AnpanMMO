# エンジンについて
2018/12/23現在、エンジンはEpicGamesのプロジェクトから落としてきたソースコードをビルドしたものを使用しています。  
プロジェクトを開きたい方は、以下の手順を踏んでいただくようお願いします。  
1.EpicGamesのリポジトリからエンジンソースコードをClone。（Cloneするまでの手順については各自でググってください。）  
2.Cloneしたソースコードをビルド。（時間がかかります。）  
3.AnpanMMO.uprojectを右クリックし、「Switch Unreal Engine Version」を選択。  
4.エンジンをインストールしたディレクトリが選択できるようになっているのでそれを選択。  
5.slnファイルをビルド。  

※１：リポジトリ作成者はEpicGamesからエンジンコードをForkしてからビルドしています。  
場合によってはエンジンのソースコードに手を加える可能性がありますので注意してください。  
（最も、その場合はEpicGamesのリポジトリにPullRequestを投げるようにはしますが・・・。極力オレオレエンジンには改造しないようにします。）  
２：「このブランチを見てね」と言った指定は特にありません。（EpicGamesのリポジトリにあった過去のPullRequestを見た感じ、PullRequestがマージされるブランチは特に指定されている訳ではなさそうなので。）
