# AutomationToolのパスを通す
ＤＬＣのビルドにはAutomationToolを使います。  
その為、まずはAutomationTool.exeへのパスを登録する必要があります。  
ウィンドウの上側にある「...」ボタンを押下するとダイアログが開くので、エンジンのAutomationTool.exeを選択してください。  
({EngineDir}/Engine/Binary/DotNET/AutomationTool.exeです。)  

# ＤＬＣを生成
チェックボックスから生成したいＤＬＣにチェックを入れて、「ＤＬＣ作成」ボタンを押下してください。  
エラーが無ければ、暫くすればＤＬＣが生成されます。  

# ＤＬＣ転送先の追加
ＤＬＣを転送する前に、転送先の情報を追加する必要があります。  
「転送先の追加」ボタンを押下するとダイアログが表示されるので、以下の必要な項目を入力してください。  
名称：転送先の名称。任意なので分かりやすい名前でＯＫです。  
ホスト：転送先のホスト。  
ユーザ名：SSHでログインするユーザ名。  
パスワード：SSHでログインするためのパスワード。  
転送先ディレクトリ：転送先のディレクトリ。（大抵public_html以下にディレクトリを作成してそこに転送する事になると思います。）  
上記の必要な項目を入力後、「ＯＫ」を押下してください。

# ＤＬＣの転送
転送先リストから転送先を選択し、「転送」ボタンを押下してください。  
ＤＬＣの転送が始まります。
